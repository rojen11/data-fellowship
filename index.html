2021 Data Literacy Fellowship

We are pleased to announce that applications are open for Data Literacy Fellowship. The objective of this fellowship organized by Code for Nepal and Open Knowledge Nepal is to help underprivileged youth and those who are impacted by COVID-19 increase their access to the competitive tech market. Youths from disadvantaged backgrounds, especially Provinces 2 and 6 are highly encouraged to apply.

We are grateful to DataCamp for their support in making this Fellowship possible.

Photo by Markus Spiske on Unsplash

About the scholarship program

The Data Literacy Fellowship by Code for Nepal and Open Knowledge Nepal includes:

  1. Access to DataCamp to take courses and earn certificates.
  2. Opportunity to publish 2-4 data visualizations and blog posts via organizers website
  3. A four-week virtual internship program with Open Knowledge Nepal and partners.
  4. Profile building and boosting via the organizer network.
  5. A digital certificate from Open Knowledge Nepal and Code for Nepal after the completion of the fellowship.

Learning outcomes of the fellowship

  1. Technical skills: Data, open-source, and web development
  2. Soft skills: Marketing, public speaking, and negotiation skills.
  3. Networking and recognization
  4. Project Management and portfolio building

Personal Development Training session includes

  • Session 1: Public Speaking
  • Session 2: Financial Literacy
  • Session 3: Negotiation for beginners
  • Session 4: Branding 101
  • Session 5: Emotional and Mental Wellness
  • Session 6: CV and Resume Writing

Application Details

  • Application Open: February 5, 2021
  • Deadline to Apply: Rolling till March 30, 2021
  • Number of Fellows for this intake: 125

Applicants must meet the following criteria to apply

  1. Able to complete the program
  2. High school degree or above
  3. Background in basic programming

We will call for interviews if you are shortlisted. The selected fellows must complete all the phases of the program to receive certification.
